# Short-Term Inputs

This folder stores the input datasets used in the short-term dispatch model.

Files in this folder include:
- `prices_2024_30min_plus_h2.csv`
- `region_absorb_need_curve_30min.csv`

Optional reference file:
- `national_absorb_need_curve_30min.csv`

These files are used in the short-term MILP model to simulate how batteries, power-to-heat, and electrolysis absorb regional renewable surplus under different project scales.
