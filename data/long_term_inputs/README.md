# Long-Term Inputs

This folder stores the input datasets used in the long-term optimization model.

Files in this folder include:
- `prices_2024_30min_elec_plus_h2_fixed2024_margin1.0.csv`
- `FR_greenH2_cost_from_hourly_power_price.csv`
- `Yearly_data.csv`
- `Material_price_350bar.csv`
- `Material_price_700bar.csv`
- `Material_consumption_350bar.csv`
- `Material_consumption_700bar.csv`

These files are used for annual optimization, techno-economic assessment, and hydrogen storage cost estimation in the long-term port-based infrastructure model.
