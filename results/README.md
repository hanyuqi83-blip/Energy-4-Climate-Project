# Results

This folder contains the main outputs generated by the project, including curtailment-risk identification results, absorb-need allocation results, short-term dispatch results, and long-term optimization results.

## Structure

### `curtailment_risk/`
This subfolder stores the outputs of the curtailment-risk identification stage, including event blocks, union hours, regional export-risk points, and summary statistics.

### `absorption_allocation/`
This subfolder stores the outputs of the absorb-need allocation stage, including national and regional absorb-need curves and allocation summaries.

### `short_term_dispatch/`
This subfolder stores the outputs of the short-term dispatch model under different project scales.

### `long_term_optimization/`
This subfolder stores the outputs of the long-term optimization model for the port-based hydrogen infrastructure system.
